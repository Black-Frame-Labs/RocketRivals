# Privacy Policy - Rocket Rivals Last updated: 2026-01-01 Rocket Rivals is a local multiplayer arcade game developed by BlackFrameLabs. We respect your privacy and aim to collect as little information as possible. ## Information We Collect Rocket Rivals does not require user accounts and does not intentionally collect personal information such as: * Real names * Email addresses * Phone numbers * Contacts * Photos * Messages * Precise location data Player names shown in-game are randomly generated locally on the device. ## Local Multiplayer Communication Rocket Rivals uses nearby device communication technologies such as Bluetooth and local peer-to-peer networking for multiplayer gameplay. This communication is used only for gameplay synchronization between nearby devices. We do not operate online game servers. ## Analytics and Crash Reporting Rocket Rivals uses Google Firebase services including: * Firebase Analytics * Firebase Crashlytics These services may collect anonymous technical and gameplay information such as: * App startups * Game mode usage * Hosted multiplayer sessions * Single-player game starts * Arena identifiers * Difficulty selection * Device type * Operating system version * Performance information * Crash logs Rocket Rivals currently sends the following anonymous analytics events: ### app_startup * Platform type * Target platform * Startup timestamp * Anonymous session identifier * Anonymous client identifier ### game_hosted * Multiplayer mode * Lobby capacity * Anonymous session identifier * Anonymous client identifier ### single_player_start * Arena identifier * Difficulty setting * Anonymous session identifier * Anonymous client identifier Rocket Rivals does NOT intentionally collect or transmit: * Real names * User-written text * Contacts * Messages * GPS/location data * Player codenames * User-created arena names * Gameplay chat * Multiplayer peer identifiers Analytics can be disabled from the in-game Settings menu. We do not use analytics for advertising personalization. We do not sell user data. Please note that third-party providers such as Google may automatically receive standard network metadata such as IP address, device information, or user agent information as part of normal HTTPS communication. For more information about Firebase privacy practices, visit: https://firebase.google.com/support/privacy ## Version Checking Rocket Rivals may periodically download a small version information JSON file from the official Rocket Rivals website in order to: * Check the latest available version * Inform users about updates * Display release information This request does not intentionally include personal information. ## Children’s Privacy Rocket Rivals is designed to avoid collecting personal information from children. If you believe personal information has been unintentionally collected, please contact us and we will remove it if reasonably possible. ## Third-Party Services Rocket Rivals may use the following third-party services: * Google Firebase Analytics * Google Firebase Crashlytics * GitHub Pages These services may process technical data according to their own privacy policies. ## Data Security We aim to minimize data collection and use reasonable technical measures to protect the limited technical data processed by the application. However, no method of electronic transmission or storage is completely secure. ## Your Rights Depending on your location, you may have rights regarding your personal data under laws such as GDPR. Because Rocket Rivals intentionally avoids collecting personally identifiable information, our ability to identify or associate technical data with a specific individual may be limited. ## Changes to This Policy This Privacy Policy may be updated from time to time. Updated versions will be published on the official Rocket Rivals website. ## Contact Developer: BlackFrameLabs Contact: blackframelabs.dev@gmail.com Website: https://black-frame-labs.github.io/RocketRivals/
